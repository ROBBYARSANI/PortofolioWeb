Vein Detection Using Noir Camera

Vein Detection Using NoIR is a computer vision project that identifies human veins by leveraging a NoIR camera and the OpenCV library. The system applies the CLAHE (Contrast Limited Adaptive Histogram Equalization) method to enhance image contrast, enabling veins to be captured and visualized clearly. This project demonstrates effective implementation of image processing and computer vision techniques for real-time biomedical detection.

Analog Line Tracer

Analog Linetracer is a fully analog line-following robot designed to track and navigate along a path using 6 infrared sensors. The system features a comparator circuit to control the left and right motors, a PWM circuit to regulate speed, and a decoder circuit to determine automatic stopping behavior. Built without microcontrollers, this project highlights precise analog control, sensor integration, and motor response tuning for consistent and stable line tracing performance.
